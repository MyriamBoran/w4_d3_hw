Here is some sample text about display types. The three most common display types are block, inline, and inline-block. A block element like a div takes up the whole width of its container, regardless of how wide you set the element to be. An inline element like a text span or a link only takes up the space it needs, and other elements can wrap around it. An inline block element is like an inline element but can have extra properties added to it that an inline element would disregard. Look at this button: - it has a width and height added to it.
